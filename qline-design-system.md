# Flowgate Design System

This document outlines the core design tokens and guidelines for the Flowgate project.

## Philosophy: Warm Industrial Precision
Confident typography, deliberate negative space, warm neutrals with a sharp accent color, and subtle grain textures that add depth.

## Typography
- **Display Font:** `DM Sans` (Clean, geometric, modern)
- **Body Font:** `Plus Jakarta Sans` (Highly legible, warm, professional)

## Color Palette

### Light Mode
- **Background:** `#FAFAF8` (Warm off-white)
- **Foreground:** `#1C1C1A` (Warm charcoal)
- **Primary (Teal):** `#0D9488` (Teal-600)
- **Accent (Amber):** `#F59E0B` (Amber-500)
- **Muted / Secondary:** `#F0EFEB` (Warm beige)
- **Border:** `#E5E4DF` (Warm border)

### Dark Mode
- **Background:** `#111110` (Warm near-black)
- **Foreground:** `#ECECEA` (Warm light)
- **Primary (Teal):** `#2DD4BF` (Teal-400)
- **Accent (Amber):** `#FBBF24` (Amber-400)
- **Muted / Secondary:** `#1A1A18` (Warm dark)
- **Border:** `#2A2A27` (Warm dark border)

## Styling Patterns
- **Radius:** `0.375rem` (6px) for interactive elements, `1rem` (16px) for cards.
- **Grain Overlay:** Global SVG turbulence noise at low opacity (`0.025` to `0.045`) for visual grit and premium atmosphere.
- **Dot Grid:** Subtle radial dot grids (`radial-gradient(border 1px, transparent 1px)`) as section backdrops.
- **Shadows:** Warm-tinted charcoal shadows rather than cold gray.
